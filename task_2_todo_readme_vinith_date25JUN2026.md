# Todo Application

A simple and user-friendly Todo Application that helps users organize and manage their daily tasks efficiently.

---

# Project Overview

The **Todo Application** is designed to help users keep track of their daily activities. Users can create, update, delete, and mark tasks as completed.

This project demonstrates basic CRUD (Create, Read, Update, Delete) operations and provides an easy way to manage personal tasks.

---

# Features

- Add new tasks
- Edit existing tasks
- Delete unwanted tasks
- Mark tasks as completed
- View all tasks
- View completed tasks
- View pending tasks
- Responsive and user-friendly interface

---

# Installation Guide

## Prerequisites

- Node.js
- npm (Node Package Manager)

## Steps

### 1. Clone the repository

```bash
git clone https://github.com/vinith33/todo-application.git
```

### 2. Navigate to the project folder

```bash
cd todo-application
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the application

```bash
npm start
```

The application will start on:

```text
http://localhost:3000
```

---

# Usage Instructions

1. Open the application in your browser.
2. Enter a task in the input field.
3. Click the **Add Task** button.
4. View the task in the task list.
5. Click **Edit** to modify a task.
6. Click **Complete** to mark the task as finished.
7. Click **Delete** to remove the task.
